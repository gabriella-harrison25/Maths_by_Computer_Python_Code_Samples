# Assignment 1 Overview
Contains:
- *AssignA_Taylor_Series_Approx.ipynb*: Calculates and visualizes numerical polynomial approximations to sine function.
- *AssignA_Monte_Carlo_Integrals.ipynb*: Performs a Monte Carlo Estimate to a specified definite integral.
- *AssignA_Monte_Carlo_Volume.ipynb*: Performs a Monte Carlo Estimate to find volume of a simple cylinder.
- *AssignA_Dataset_Histograms.ipynb*: Generates a histogram based on imported Wine Quality Dataset.
- *wine+quality (1).zip*: Copy of Wine Quality Dataset from UCI Machine Learning Repository. Required for *AssignA_Dataset_Histograms.ipynb* above.
